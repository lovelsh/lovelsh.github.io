## 자주 묻는 질문

### Q. 새 일정은 어떻게 등록하나요?

- 첫 화면(일정 탭)에서 우측 상단 **+** 버튼을 눌러 제목, 내용, 날짜·시간, 카테고리(학교 / 학원 / 숙제 / 행사 / 준비물 / 현장학습 / 상담 / 기타), 자녀를 입력합니다.
- 같은 화면에서 **사진 선택** 또는 **카메라** 버튼으로 알림장·안내문 사진을 한 번에 여러 장 첨부할 수 있습니다.
- 저장하면 일정 목록과 캘린더 양쪽에 즉시 반영됩니다.

### Q. 자녀를 여러 명 관리하려면 어떻게 하나요?

- **설정 → 자녀 관리 → 자녀 목록** 에서 자녀를 추가하세요. 자녀마다 이름·어린이집(학교) 이름·색깔을 지정할 수 있습니다.
- 일정 목록 / 캘린더 상단의 자녀 칩(전체·자녀별)을 눌러 특정 자녀의 일정만 필터링할 수 있습니다.
- 자녀별 색깔은 일정 행 좌측 막대 / 캘린더 점 표시에 그대로 사용되어 한눈에 구분됩니다.

### Q. 알림은 언제 오나요?

- 일정 등록 시 **알림 토글**을 켜면, **1일 ~ 100일 전** 중에서 며칠 전 알림을 받을지 선택할 수 있습니다.
- 알림 시각도 함께 지정합니다(예: 등원 전 오전 8시).
- 알림이 오지 않는 경우 **iOS 설정 → 알림 → 자녀일정** 에서 "알림 허용", 배너 / 사운드 / 뱃지가 모두 켜져 있는지 확인하세요.
- 알림은 기기 내부에서 예약되는 로컬 알림이므로 인터넷 연결이 없어도 정상 동작합니다.

### Q. 홈 화면 위젯은 어떻게 추가하나요?

- 홈 화면 빈 공간을 길게 눌러 좌측 상단 **+** 버튼을 탭한 뒤, 검색창에 `ScheMng` 또는 `자녀일정`을 입력합니다.
- 작은 사이즈 / 중간 사이즈를 지원합니다. 위젯에는 가장 가까운 다가오는 일정 한 건이 자녀 이름·날짜·카테고리 아이콘과 함께 표시됩니다.
- 위젯은 일정 추가 / 수정 / 삭제 시 자동으로 갱신되지만, 실제 새로고침 시점은 iOS 가 결정하므로 짧은 지연이 발생할 수 있습니다.

### Q. 알림장 사진을 첨부하면 어디에 저장되나요?

- 첨부된 사진은 앱 전용 저장 영역(앱 샌드박스 내 `ScheduleImages` 폴더)에 JPEG 로 저장됩니다.
- 외부 서버로 업로드되지 않으며, 운영자는 사진에 접근할 수 없습니다.
- 일정 상세 화면에서 사진을 탭하면 확대 보기로 알림장 내용을 자세히 확인할 수 있습니다.

### Q. 백업과 복구는 어떻게 하나요?

- **설정 → 데이터 관리 → 백업 내보내기** 로 모든 자녀 / 일정 / 사진 데이터를 `.schemng` 백업 파일로 내보낼 수 있습니다 (iCloud Drive / AirDrop / 메시지 등으로 전송 가능).
- **백업 복구** 로 백업 파일을 선택한 뒤 **덮어쓰기** (기존 데이터 삭제 후 백업 데이터로 교체) 또는 **병합** (기존 + 백업 데이터 모두 유지) 중에서 선택할 수 있습니다.
- 기기 변경 / iOS 재설치 / 실수 삭제 시 백업 파일이 있으면 데이터를 복구할 수 있습니다.

### Q. 완료된 일정은 어떻게 정리하나요?

- 일정 행을 좌→우로 스와이프하면 **완료 / 미완료** 처리가 가능합니다. 또는 행 좌측 동그라미를 직접 탭해도 됩니다.
- 일정 목록에서 **전체 / 완료 / 예정** 세그먼트 컨트롤로 필터링할 수 있습니다.
- "다가오는 일정 / 지난 일정 / 날짜 미정" 구간으로 자동 분리되어 표시됩니다.

### Q. 광고는 왜 보이나요?

- ScheMng 의 1.0 은 **무료 + Google AdMob 배너 광고** 모델입니다. 메인 화면 하단에 50pt 배너가 한 줄 표시됩니다.
- 첫 실행 시 앱 추적 투명성(ATT) 다이얼로그가 표시됩니다 — 거부해도 앱 이용에는 어떠한 제한도 없습니다.
- 광고 제거 인앱 결제 도입은 후속 업데이트로 검토 중입니다.

### Q. 어떤 iOS 버전이 필요한가요?

- iOS 17 이상이 필요합니다. SwiftUI / SwiftData / iOS 17+ 위젯·로컬 알림 API 를 사용합니다.

### Q. 모든 데이터를 삭제하려면 어떻게 하나요?

- 일정 상세 화면에서 개별 삭제할 수 있습니다.
- 자녀 단위 삭제는 **설정 → 자녀 관리 → 자녀 목록** 에서 가능합니다. 자녀를 삭제하면 그 자녀에게 연결된 모든 일정과 사진이 함께 삭제됩니다.
- iPhone 에서 자녀일정 앱을 삭제하면 기기 내부 데이터도 모두 사라집니다.
