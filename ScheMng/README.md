## 앱 소개

ScheMng(자녀일정)는 어린이집·유치원·학교에서 받아 온 알림장 사진을 일정에 그대로 첨부해 두고, 카테고리(학교 / 학원 / 숙제 / 행사 / 준비물 / 현장학습 / 상담 / 기타)와 자녀별 색깔로 정리한 뒤, 며칠 전 로컬 알림과 홈 화면 위젯으로 자녀의 행사·준비물·현장학습 일정을 잊지 않게 챙겨 주는 iPhone 전용 앱입니다.

- iOS 17 이상 (SwiftUI / SwiftData / WidgetKit 기반)
- 한국어 단독 출시 (1.0)
- 자녀 데이터는 모두 기기 내부에 저장 — 외부 서버로 전송되지 않습니다
