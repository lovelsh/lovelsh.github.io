# BusJam — 출퇴근 길 짧은 휴식

Welcome / 환영합니다.

**BusJam** 은 출퇴근 버스나 지하철에서 잠깐이라도 푹 쉬고 싶을 때 켜는 짧은 수면·휴식 보조 앱입니다. 알파파(약 10 Hz) → 세타파(약 6 Hz) → 알파파 순서의 양이청음(binaural beats) 으로 짧은 시간 안에 휴식 모드로 들어가도록 돕고, 마지막에는 부드럽게 깨어나도록 정리해 줍니다. 취향대로 자연 배경음(파도 / 바람 / 비 / 새 / 시냇물 / 지하철 / 백색소음 / 모닥불 / 폭포)을 골라 본인만의 휴식 트랙을 만들어 보세요.

> BusJam is a commute-friendly short-rest companion. Alpha → Theta → Alpha binaural beats plus a nature-sound mixer (waves, wind, rain, birds, stream, subway, pink noise, campfire, waterfall) help you drop into a calm rest state and wake up gently before your stop.
